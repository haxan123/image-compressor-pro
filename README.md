Compress JPEG and PNG images directly on your server. Reduce file sizes by up to 80% without losing quality.

== Description ==

= Features =

* ✅ Local Compression - All compression happens on your server
* ✅ JPEG & PNG Support - Compress both formats
* ✅ Bulk Optimization - Optimize all images with one click
* ✅ Auto Compression - Automatically compress images on upload
* ✅ Backup & Restore - Keep originals, restore anytime
* ✅ Quality Control - Adjust compression quality (50-100%)
* ✅ Media Library Integration - Compress individual images
* ✅ No API Keys - Works out of the box

= How It Works =

1. Install and activate the plugin
2. Go to Image Compressor in your admin menu
3. Adjust compression quality (85% recommended)
4. Click "Start Bulk Optimization" or let it work automatically

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/image-compressor-pro/` directory, or install through WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to Image Compressor menu to configure settings
4. Adjust compression quality as needed
5. Click "Start Bulk Optimization" to compress existing images

== Frequently Asked Questions ==

= Does this plugin send my images to external servers? =

No. All compression happens locally on your server. Your images never leave your hosting environment.

= Will this reduce image quality? =

You control the compression quality. At 85% quality (default), the difference is barely noticeable to the human eye but file sizes reduce significantly.

= Can I restore original images? =

Yes. The plugin automatically creates backups of original images. You can restore any image at any time.

= What image formats are supported? =

Currently supports JPEG and PNG formats. WebP support coming soon!
